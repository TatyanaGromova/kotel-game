# КотёлЪ: миссия «Тёплая зима»

Интерактивная промо-игра сервисного центра «КотёлЪ» — 5 уровней, тепло-баллы и зимний бонус до 2000 ₽.

## Стек

- React 18 + TypeScript
- Vite 6
- Tailwind CSS 3
- Framer Motion
- Lucide React

## Запуск

```bash
npm install
npm run dev
```

Сборка для встраивания на сайт или VK Mini Apps:

```bash
npm run build
```

Артефакты в `dist/`. Базовый путь `./` — можно разместить в подпапке.

## Структура

- `src/data/` — уровни, вопросы, награды, населённые пункты (редактируются без правки логики)
- `src/components/` — экраны и визуальные эффекты
- `src/levels/` — игровая логика уровней
- `src/hooks/useGameStorage.ts` — прогресс в `localStorage`

## Прогресс

Ключ: `kotel-warm-winter-v1`. Кнопка «Сбросить прогресс» внизу страницы.

## Интеграция

- **Сайт:** iframe или статика из `dist/`
- **Отдельная ссылка:** деплой `dist/` на любой хостинг
- **VK Mini Apps:** собрать `dist`, подключить как WebView; при необходимости заменить ссылки в `FinalScreen.tsx`
